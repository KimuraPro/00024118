Website Purpose

The Kitchen Table is an educational website dedicated to the art and science of cooking. It aims to provide visitors with curated recipes, essential cooking techniques, and an exploration of world cuisines. The site demonstrates good usability practices, accessible design, and responsive layouts across devices.

All content has been gathered and organised for educational purposes. The website follows accessibility guidelines (WCAG) including proper colour contrast, semantic HTML structure, descriptive alt text for images, and keyboard-navigable menus.

Design Principles

  • Usability: Clear navigation structure with consistent header and footer across all pages.
  • Accessibility: Semantic HTML5 elements, ARIA labels, descriptive alt text, and sufficient colour contrast.
  • Responsiveness: Mobile-first design using CSS flexbox and grid, ensuring readability on all screen sizes.
  • Visual Hierarchy: Distinctive heading fonts (Playfair Display) paired with readable body text (Source Sans 3).
